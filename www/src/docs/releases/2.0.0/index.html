Release Version 2.0.0 (Latest: 2.0.2) The following changes were made in version 2.0.0 of Thunderous:
  • Removed deprecated type RenderProps in favor of RenderArgs.
  • Added getter() method to RenderArgs. Read more about getters.
  • Deprecated the customCallback() method, which was used to register callbacks on the class instance. Now, callbacks are now bound directly to DOM nodes. Read more about event binding.
  • Removed scoped custom element registry polyfill from the library. This must now be included separately, as an optional peer dependency. Read more about registries.
